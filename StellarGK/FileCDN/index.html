How do I get in?

Quick version: if you have version 1.033.12 of the game and a rooted Android device or emulator, open a Terminal and enter these commands:

u0_a40@android:/ $ su
root@android:/ # mount -o rw,remount /system
root@android:/ # echo 5.152.179.136 gk.flerogames.com>/etc/hosts
root@android:/ # cat /etc/hosts
5.152.179.136 gk.flerogames.com
root@android:/ #

If the cat command returns that line, you have correctly modified the hosts file; you can start the game and log in. If your commander is level 89, you're in.

If all that went over your head, read the rest of the page for detailed instructions.

How do I get out?

Same as above, but use the command echo 127.0.0.1 localhost>/etc/hosts instead. That removes the server redirection. Make sure you exit the game completely first!

Why do I need a rooted device or emulator?

In order to use the original game client without any modifications, you need to fool it into connecting to GKServer instead of the official server. To do that, you have to modify the "hosts" file, which tells your device or emulator what the IP address of a server is. Modifying this file requires root access.

If you don't have a rooted device, use an Android emulator. Several of them have options to enable rooting; Nox App Player has options to enable root access by changing "Root Startup" to "Enable" in their settings.

To install the game in the emulator, just go to the included Google Play store, log in, and download the game as usual. You can also install it from a different App Store or sideload the 1.033.12 version of the file if you don't want to use the Google Play store; as long as it's the original, unmodified game install, it will work.

What is a Terminal and how do I use it?

A terminal is an app that lets you give commands to the device or emulator in text-mode. It's mainly used by advanced users to do things that are not possible with apps; like modifying the hosts file. Any Terminal app will do; again, search the App Store and you'll see dozens. Try to find one that doesn't contain ads, but you're only going to use it for five minutes and you can remove it later, so it doesn't really matter.

After installing the Terminal, open it. You'll be greeted by a text-only window with a line that ends in "$". Type su, followed by the Enter key, to enable root mode. The device or emulator will ask you to confirm this; allow it. If the Terminal says that the command is invalid, your device is not rooted. You need a rooted device to do this.

If your terminal now shows a line that ends with "#", you are in root mode. Type the command mount -o rw,remount /system, followed by the Enter key, to enable writing to the hosts file; then type echo 5.152.179.136 gk.flerogames.com>/etc/hosts, followed by the Enter key, to redirect the game server to the GKServer IP. If you get no errors up to this point, you're done; but just to confirm that it worked, type the command cat /etc/hosts, followed by the Enter key. It should display the line you just typed, like in the sample at the top of the page.

If, at any point, the keyboard stops working in your Terminal app, you probably have one that is stuck in soft keyboard mode. Click the three dots and the top right and select "Toggle Soft Keyboard" to fix it.

What game version do I need?

GKServer supports version 1.033.12 of the game, the latest at the time of this writing. It shows up in the bottom-left corner of the game's title screen, and in the bottom-right when you click on your portrait in-game and go to "Game Info".

If the version that your app store installed is newer than that, you'll need to download 1.033.12 from somewhere else, like apkplz.com, always making sure it's the unmodified version of the game. You do not need a "mod" APK, just the original.

After installing the APK in your device or emulator, make sure to copy the .obb file that comes with it into /sdcard/Android/obb/com.flerogames.GK or the game will not start. Both the folder and the .obb filenames must be all lowercase except "GK" in uppercase, or the game will not start.

The game assumes you're playing in the Global server. If you use the Korean server, it will still work, but the three Korean pilots will be locked.

What Works

Portrait. If you click your portrait at the top left of the screen, you can rename your character and select the girl you want to show up in the map screen. Turning off the sound, music and skill animations also works fine, because that doesn't talk to the server.

Headquarters. From here you can see your pilots, change their costumes, and set up their Ooparts. All the girls are Level 89, Gold+4 and have maximum Skills and Affection; support for setting your own levels is coming very soon.

Mail. If you tap the green box in the top right of the game's screen, you will have one email containing 99 Oopart boxes. Claim it if you want to slot Ooparts.

Storage. Here you should see your 99 Ooparts boxes. Opening them gives you 50 of each Oopart; even if you open just one box. The amount of boxes you open at once determines what level of Ooparts you get; if you want level 10 Ooparts, open 10 boxes. If you want level 99 Ooparts, open all 99 boxes.

Federation. Not much to see here other than see the Federation stats; it's max level with all the skills maxed out; support for setting your own Federation levels is coming soon.

Treasurse. This is a proper simulator, using the same drop rates as the game. It does not currently keep track of how many fails until a guaranteed pilot; coming soon.

Bank. This is a proper simulator, using the same drop rates as the game.

Boss Raid. You get a random boss every time you enter this screen. You can go ahead and fight, but the final screen does not report your final score; you have to keep track of it during the fight, or use the "Battle Results" button and add up the "Enemy Damage" totals.

Daily Battle. All three verions are available and you can fight any of them. The final ones will even challenge your OP units.

Death Match. You will be at stage 21, and face Jung by default. If you enter the Shop, you can "buy" the boss you want to fight. After buying the boss, you need to leave and enter the Death Match screen for it to update. All the commanders are available and have maxed SP.

Shootout Arena. The bullet shootout is always available here. It doesn't keep track of rewards, but you can try various formations to see how far you get.

Arena. Both 1v1 and 3v3 modes has three fights randomly picked from Global server, so you can test OP formations. They have all been buffed to the same level as you.

World Map. All stages are unlocked; pick one and fight.

This is a work in progress. If you try to access things that aren't listed in here, the game will give an error code and exit. It won't hurt anything, but you will need to log in again, and no amount of retrying will make those things work until they are listed above.
